# AI-Powered Gmail Automation using n8n 

##  Overview

This project is an AI-powered email automation workflow built using **n8n**, **Google Gemini AI**, and **Gmail**. The workflow automatically generates professional email content based on user input and sends the email through Gmail without manual drafting.

This project demonstrates workflow automation, AI integration, prompt engineering, and business process automation using a low-code platform.

---

## Features

* Automated email generation using Google Gemini AI
* Professional email formatting
* Gmail integration for automatic email delivery
* Low-code workflow automation with n8n
* End-to-end AI-powered email workflow

---

## Technologies Used

* n8n
* Google Gemini AI
* Gmail API
* JSON Workflow Configuration

---

##  Workflow Process

1. User provides an email request or prompt.
2. Gemini AI generates professional email content.
3. n8n processes the generated response.
4. Gmail automatically sends the email to the recipient.
5. The email is recorded in the Sent mailbox.
